# Project Objective
To analyze 51,000+ global retail transactions across countries, regions, customer segments, and product categories — 
identifying sales trends, profitability patterns, and the impact of discounting on business performance through an interactive 
Excel dashboard.

# Key Business Questions
Which countries are driving the highest sales and which are generating losses?
How are sales and profit trending month-over-month and year-over-year?
What impact are discounts having on overall profitability across discount bands?
Which sub-categories are most and least profitable?
How are loss-making orders distributed across product categories?
Which ship modes contribute most to overall sales?

# Process
Data Cleaning — Cleaned and standardized 51,000+ records using Power Query
Data Modelling — Built structured data model in Power Pivot for accurate cross-table calculations
DAX Measures — Created calculated KPIs including YoY growth, average delivery days, profit margin, and country-
level performance metrics
Dashboard Design — Built interactive visualizations using Pivot Tables, slicers across 6 filters — Category, Order Status, 
Segment, Region, Ship Mode, and Order Priority

# Key Insight
Despite 24% year-on-year revenue growth, high-discount orders account for the majority of total losses — revealing that 
revenue growth and profitability are not always moving in the same direction.
